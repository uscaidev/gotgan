# Linkage Assessment

## 판정 3축

| 축 | 판정 재료 (인덱스 필드) | 강등/차단 규칙 |
|---|---|---|
| 갱신주기 | 업데이트 주기, 차기 등록 예정일 | 연/분기 갱신 + 실시간 요구 → 배치로 강등 권고 |
| 망환경 | (사용자 제공) 인터넷망/폐쇄망 | 폐쇄망 → API 직접 호출 차단, 망연계 중계 or 파일 반입 |
| 트래픽 | 신청가능 트래픽, 심의 유형 | 개발계정 한도로 운영 설계 → 오류 지적, 심의승인 → 리드타임 경고 |

## 결합키 우선순위 (find_joinable 예정)

1. 행정표준코드(법정동코드·행정기관코드) — 최상급
2. 표준데이터(목록유형=표준, 스키마 고정) — 기관 간 수평결합
3. 분류체계·기관 계열 유사 — 약한 후보, low_confidence 필수 표기
