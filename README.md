# Gotgan

Gotgan is a deployment-ready repository for the 곳간 project.

곳간은 흩어진 정보와 작업 흐름을 한곳에 모아, 필요한 자료를 빠르게 찾고 정리할 수 있도록 준비하는 프로젝트입니다. 이 저장소는 서비스 배포를 위한 코드, 설정, 문서가 쌓일 기본 공간으로 사용됩니다.

## Purpose

- 프로젝트 소개와 배포 준비 과정을 정리합니다.
- 초기 소스 코드와 운영 설정을 안전하게 관리합니다.
- 이후 기능 개발, 문서화, 배포 자동화를 위한 기준점을 만듭니다.

## Repository Status

This repository has just been initialized. Application source, deployment configuration, and usage documentation will be added as the project takes shape.
